### 🔴 P0 - 严重错误（必须立即修复）

#### 问题1：界面能量方程扩散焓项符号错误

**位置**: `physics/interface_bc.py:387`

**问题描述**：
```python
# 当前代码（错误）
rhs = q_diff_species_pow  # Line 387

# 方程展开为：
# (k_g/dr_g) * Tg + (k_l/dr_l) * Tl - (k_g/dr_g + k_l/dr_l) * Ts - L_v * mpp = q_diff_species_pow
# 移项后：
# ... - q_diff_species_pow - L_v * mpp = 0
```

**设计文档要求**（符号约定.md:178-180）：
```
q_l·n - q_g·n + Σ h_k,g J_g,k·n - m'' h_vap = 0
```

**正确形式应为**：
```python
# 修正方案1：修改rhs符号
rhs = -q_diff_species_pow  # 负号！

# 或方案2：修改方程结构
coeff_mpp = -A_if * L_v
rhs = -q_diff_species_pow  # 扩散焓项移到右侧时需要负号
```

**影响**：能量平衡错误，多组分蒸发时Ts预测不准确

**修复优先级**：🔴 **最高（P0）**

#### 问题2：线性模式实际为半隐式，而非全隐式

**位置**: `solvers/timestepper.py:116-151`

**问题描述**：
```python
# 线性模式（Line 116-151）
state_guess = state_old  # Line 130
props_old = props  # Line 114 (基于state_old计算)

A, b, diag_sys = _assemble_transport_system_step12(
    state_guess=state_guess,  # 使用state_old
    props=props_old,          # 使用基于state_old的物性
    ...
)
```

**实际求解**：`A(u^n) @ u^{n+1} = b(u^n)` → **半隐式**

**设计文档要求**：`A(u^{n+1}) @ u^{n+1} = b(u^{n+1})` → **全隐式**

**对比非线性模式**（正确的全隐式）：
```python
# residual_global.py:112-120
state_guess = ctx.make_state(u)  # 每次Newton迭代都更新
props = compute_props(cfg, grid, state_guess)  # 重新计算物性
A, b = build_transport_system(..., state_guess=state_guess, props=props)
```

**影响**：
- 线性模式收敛性和精度低于全隐式
- 设计文档未明确区分两种模式
- 可能误导用户认为已实现全隐式

**修复建议**：
1. **方案A（推荐）**：在文档中明确说明两种模式的区别
   - `nonlinear.enabled=false`：半隐式（MVP/测试用）
   - `nonlinear.enabled=true`：全隐式（生产用）

2. **方案B**：修改线性模式为简单不动点迭代（1次迭代的全隐式）
   ```python
   # 在线性模式中添加：
   props_guess = compute_props(cfg, grid, state_old)
   for iter in range(1):  # 简单1次更新
       A, b = assemble(..., props=props_guess)
       u_new = solve(A, b)
       state_new = unpack(u_new)
       props_guess = compute_props(cfg, grid, state_new)
   ```

**修复优先级**：🔴 **高（P0）**

#### 问题4：后处理修改解破坏残差一致性

**位置**: `solvers/timestepper.py:196-199, 358-361, 519-562`

**问题描述**：
```python
# 1. mpp非负约束（Line 196-199）
if state_new.mpp < 0.0:
    state_new.mpp = 0.0  # 在残差外部强制修改

# 2. 物种边界夹紧（Line 519-562）
state.Yg[k_full, :] = np.maximum(state.Yg[k_full, :], min_Y)
state.Yg[:, :] = np.clip(state.Yg, 0.0, 1.0)
```

**影响**：
- 修改后的解不满足残差=0
- 破坏守恒性和能量平衡
- 非线性求解器可能不收敛

**修复建议**：
1. **mpp约束**：使用对数变换 `mpp = exp(log_mpp)`，求解log_mpp
2. **物种边界**：
   - 方案A：对数变换 `Y = exp(log_Y) / Σexp(log_Y)`
   - 方案B：投影方法（Projected Newton）
   - 方案C：惩罚项添加到残差中

**修复优先级**：🟡 **中（P1）**
